# 🔄 Units Converter

A modern and responsive Units Converter built using HTML, CSS, and JavaScript.

Convert values between different units quickly and accurately with a clean glassmorphism UI.

---

## 🚀 Features

### 📏 Length Conversion
- Meter
- Kilometer
- Centimeter
- Millimeter
- Foot
- Inch

### ⏱️ Time Conversion
- Second
- Minute
- Hour
- Day

### ⚖️ Weight Conversion
- Milligram
- Gram
- Kilogram
- Ton

### 🌡️ Temperature Conversion
- Celsius
- Fahrenheit
- Kelvin

### 💾 Data Conversion
- Bit
- Byte
- Kilobyte
- Megabyte
- Gigabyte

---

## ✨ UI Features

- Responsive Design
- Glassmorphism Effect
- Interactive Unit Cards
- Custom Dropdown Menu
- Mobile Friendly Layout
- Smooth Hover Animations
- Real-Time Unit Selection

## 🛠️ Technologies Used
